About This Project

About This Website

This website was created for IT 3203 to demonstrate web development skills using HTML, CSS, and JavaScript.

The topic of this project is the evolution of the Hypertext Transfer Protocol (HTTP). The website explains the history of HTTP, modern improvements, important concepts, and how web communication has changed over time.

Student Information

This project was developed as part of the IT 3203 Web Development course.

The website demonstrates the use of webpage structure, navigation, styling, responsive design, and interactive JavaScript features.
