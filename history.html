Origin of HTTP

HTTP was created by Tim Berners-Lee in 1989 at CERN to allow sharing of documents across the web.

HTTP/0.9

The first version of HTTP only supported simple GET requests to retrieve web pages.

HTTP/1.0

This version introduced status codes, headers, and support for different file types.

HTTP/1.1

HTTP/1.1 improved performance with persistent connections and better caching support.
